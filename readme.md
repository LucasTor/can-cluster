# CAN Cluster
### Gauge Cluster to communicate with FuelTech via CAN

Made to run on a Raspberry Pi running Armbian, and being displayed on those cheap chinese 10.3 1920x720 displays

Running the project

```
poetry install
poetry run python cluster.py
```
